## Quick Calculator

**Formula:** Total = Rate per Unit × Number of Units

**Example Calculation:**
```
Model: DELTA 2 BHK
Rate per Unit: ₹20,70,000
Number of Units: 7

Calculation:
20,70,000 × 7 = ₹1,44,90,000

In Words:
ONE CRORE FORTY FOUR LAKH NINETY THOUSAND ONLY
```

## Verification Checklist

When testing quotations, verify:
- ✅ Square footage matches model type
- ✅ Rate per unit is correct
- ✅ Quantity shows number of units ordered
- ✅ Total = Rate × Units
- ✅ Total in words matches numerical total
- ✅ Indian number format (Lakhs, Crores)
- ✅ Professional formatting with commas
